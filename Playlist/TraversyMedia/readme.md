## Folder Summary

This folder contains examples covering the main topics from the "PHP for Beginners Crash Course" by Traversy Media. Each file demonstrates a specific PHP concept or feature:

- **01_output.php** — Outputting data, comments.
- **02_variables.php** — Variables, data types, constants.
- **03_arrays.php** — Arrays: indexed, associative, multidimensional.
- **04_conditionals.php** — Conditional statements: if, else, switch, ternary operator.
- **05_loops.php** — Loops: for, while, do-while, foreach.
- **06_functions.php** — Functions, parameters, return values.
- **07_array_functions.php** — Built-in array functions.
- **08_string_functions.php** — Built-in string functions.
- **09_superglobals.php** — Superglobal variables: $_GET, $_POST, $_SESSION, etc.
- **10_get_post.php** — Working with forms, handling GET and POST data.
- **11_sanitizing_inputs.php** — Input validation and sanitization.
- **12_cookies.php** — Working with cookies.
- **13_sessions.php** — Working with sessions.
- **14_file_handling.php** — Reading and writing files.
- **15_file_upload.php** — File uploads.
- **16_exceptions.php** — Exception handling.
- **17_oop.php** — Object-oriented programming basics.
