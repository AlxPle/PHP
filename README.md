# Personal PHP Learning Repository

This repository is for my personal learning and practice with PHP.

## Goals
- Learn PHP syntax and core language constructs.
- Practice working with forms, sessions, and authentication.
- Work with databases (MySQL / PDO) and basic data modeling.
- Learn modern PHP tooling: Composer, autoloading, and project structure.
- Build a set of small projects to reinforce learning (ToDo, blog, CRUD apps).
